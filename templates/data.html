  • Python scraper for Maryland lobbying disclosure activity reports

    I wrote this web scraper and cleaned all the data that made this story and other published stories on lobbyist spending possible. Documentation of the scraper can be found above.

    I used the Python library BeautifulSoup to write a web scraper that grabs thousands of lobbying disclosure documents by looping through links and pagination to grab specific information such as bills lobbied for, who lobbied for the bills, how much lobbyists were paid and more. I then cleaned the data using both Python’s pandas library and Excel. I also visualized this data in Flourish, which was used in the story I've linked as well as other lobbying stories.


  • Llano River communities fight former oil executive’s plan for a private dam

    I requested the locations and information of all dams in Texas from the Texas Commission on Environmental Quality for this story, analyzed the data, wrote up my analyses in the story and visualized the data in a Datawrapper map.

    I used the Texas Almanac to determine the major rivers in Texas. I then used this information to perform data analysis using Python’s pandas library on this data, specifically focusing on Texas’ major rivers. I also used Python’s folium library to perform some exploratory mapping analysis of this data. I then wrote up my findings in the portion of the story underneath the ‘More than half of Texas dams are privately owned’ subhead.

    I also created a map in Datawrapper using this data. To make this map, I drew my own GeoJSON file of the Llano river because the river was not visible using datawrapper’s basic map backgrounds.
